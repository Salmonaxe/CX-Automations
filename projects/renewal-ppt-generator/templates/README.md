# Templates

Use one shared PowerPoint template for this project.

## Recommended file

- `company-template.pptx`

## Keep in the template

- Corporate theme/colors/fonts (Slide Master)
- Standard title slide layout
- Standard blank/content layout used by automated slides
- Footer placeholders if your standard requires them

## Remove from the template

- Introductory narrative slides
- Example/content slides with real text
- Old project-specific sections
- Any confidential sample data

## Why

The automation creates slides programmatically. It only needs branding/layout definitions from the template, not prebuilt story slides.

## Validation checklist

1. Open template and verify theme/fonts are correct.
2. Save as `.pptx` (if you received a `.potx`, save/convert it to `.pptx`).
3. Run one script with `--template-pptx` and confirm output branding.
4. Confirm title and blank/content layouts render correctly.
